About Class TimeTable

Class TimeTable v2.5 is a responsive Progressive Web App for organizing and tracking class schedules with offline support.

  • Open Source: This project is publicly available and open to community contributions.
  • Tech Stack: Built with HTML, CSS, and JavaScript (with custom framework).
  • Core Features: Real-time period highlighting, day navigation, swipe gestures, and timetable editing with local storage persistence.
  • PWA Ready: Includes a service worker and web manifest for installable app behavior and offline access.
  • Customization: Update courses in time-table.js or use the in-app editor to personalize periods.
